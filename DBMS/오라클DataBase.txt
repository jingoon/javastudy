현재 계정 
system, sys

DB의 객체
user(계정), table, 함수, sequence, index, trigger, procedure, ...

객체 명령어
생성은 create [객체]
수정은 alter
삭제는 drop

데이터 객체x
입력, 조회, 수정, 삭제 객체x
insert, select, update delete 크루드? CreateUpdateReadDelelte

계정을 생성할 때는 일정자격이 있는 계정에서 만든다

show user - 현재유저를 보여줌
conn ezen/ezen - 유저로그인 (빠져나가 로그인해도 됨)

유저 생성
create user ezen identified by ezen;
권한 부여
grant connect, resource, DBA to ezen;
권한 철회
revoke connect, resource, DBA from ezen;
유저 삭제
drop user ezen;

객체를 생성, 수정 , 삭제 하는 SQL문
DDL(Data Definition Language)
create, alter, drop

보안적인 요소를 담고 있는 SQL문
DCL
grant, rollback

TCL
commit

DML
insert, select, update, delete

-------*중요*-------
객체의~
* creat 생성 : create user ezen identified by ezen;
* alter 수정 
*drop 삭제 : drop user

* grant 권한 부여 : grant connect, resource, dba to ezen;
* revoke 부여 취소 : revoke connect, resource, dba from ezen;
============
데이터의~
* insert 입력
* select 조회
* update 수정
* delete 삭제

* commit 확정
* rollback 확정취소

테이블에 데이터 입력

SQL> insert into member (mid, name, job, birth) values ('m004', 'park','b', '2000-12-17');

SQL> insert into member values ('m005','kang','a','1990-10-03' );

테이블의 데이터 조회
